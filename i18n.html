多语言支持

在Github上编辑此页

基本用法

现在，Valine开始支持多语言。默认内置：

  • zh-cn
  • en
var valine = new Valine();
valine.init({
    el:'#vcomment',
    appId:'your leancloud appid',
    appKey:'your leancloud appkey',
    lang:'en' // 设置这里就ok
})

自定义语言

自定义语言只需三步：

  1. 定义localeName和localeMode
  2. 安装语言
  3. 初始化 Valine

请严格按照下面的localeMode格式书写内容:

// 1.定义localeName 和localeMode 
var localeName = 'en',
    localeMode = {
        head:{
            nick:'NickName',
            mail:'E-Mail',
            link:'Website(http://)',
        },
        tips:{
            comments:'Comments',
            sofa:'No comments yet.',
            busy:'Submit is busy, please wait...',
            again:'Sorry, this is a wrong calculation.',
            limit:'The largest number of words %d'
        },
        ctrl:{
            reply:'Reply',
            ok:'Ok',
            sure:'Sure',
            cancel:'Cancel',
            confirm:'Confirm',
            continue:'Continue',
            more:'Load More...',
            try:'Once More?',
        },
        error:{
            99:'Initialization failed, Please check the `el` element in the init method.',
            100:'Initialization failed, Please check your appId and appKey.',
            401:'Unauthorized operation, Please check your appId and appKey.',
            403:'Access denied by api domain white list, Please check your security domain.',
        },
        timeago:{
            seconds:'seconds ago',
            minutes:'minutes ago',
            hours:'hours ago',
            days:'days ago',
            now:'just now'
        }
    };

var valine = new Valine();
// 2.安装语言
valine.installLocale(localeName,localeMode)
// 3.初始化Valine
valine.init({
    el:'#vcomment',
    appId:'<APP_ID>',
    appKey:'<APP_KEY>',
    lang:localeName
})
